# SNAKE GAME

This is a modular Snake game with an interactive menu, built on Python using the Pygame framework.


## Features

- Modular architecture.
- State-based game system.
- Menu and settings screen.
- Sound and music manager.
- Countdown before game start.
- Score tracking.


## Built With

- Python 3.10.12
- Pygame


## Installation

Clone the repository:

```bash
git clone https://github.com/ezedpastore/snake-game.git
cd snake-game
```

Create a virtual environment:

```bash
python -m venv venv
source venv/bin/activate  # Linux / Mac
venv\Scripts\activate     # Windows
```

Install dependencies: 

```bash
pip install -r requirements.txt
```


## Launch the game

```bash
python3 main.py 
```


## How to play

- Arrow Keys → Move
- Enter → Select menu option
